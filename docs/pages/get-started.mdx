# Get started

create a new reeact or next.js project and install the `morocco-svg-map` package.

## Installation
```bash copy
npm install morocco-svg-map
```

or

```bash copy
yarn add morocco-svg-map
```

## Usage

```jsx filename=*.js {7} copy
import React from 'react';
import { Morocco } from 'morocco-svg-map';

const App = () => {
  return (
    <div>
      <Morocco />
    </div>
  );
};
```
